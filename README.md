## Что такое конструктор?

**Конструктор** — это специальный метод, который вызывается при создании объекта (оператор `new`).  
Он нужен, чтобы сразу задать начальные значения полей.

**Особенности конструктора:**
- Называется так же, как класс
- Не имеет возвращаемого типа (даже `void`)
- Вызывается автоматически при `new`

---

## Виды конструкторов

| Тип | Описание |
|-----|----------|
| **По умолчанию** | Создаётся автоматически, если не написан свой |
| **С параметрами** | Позволяет сразу задать значения полей |
| **Перегрузка** | Несколько конструкторов с разными параметрами |
| **this()** | Вызов одного конструктора из другого |

---

## Синтаксис

```java
class Car {
    String model;
    int year;

    // Конструктор по умолчанию
    Car() {
        model = "Неизвестно";
        year = 0;
    }

    // Конструктор с параметрами
    Car(String model, int year) {
        this.model = model;
        this.year = year;
    }
}
```
