# Студент: конструктор с параметрами

Это первый пример по конструкторам. Я создаю класс `Student` с конструктором, который принимает имя и возраст, и методом `introd()` для вывода информации.

---

## Код

```java
class Student {
    String name;
    int age;

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String introd() {
        return "Меня зовут " + name + ", мне " + age + " лет";
    }
}
```
Класс StudentMain (создание объектов)
```java
public class StudentMain {
    public static void main(String[] args) {
        Student one = new Student("Anna", 32);
        Student two = new Student("Tom", 36);
        System.out.println(one.introd() + "\n" + two.introd());
    }
}
```
**Вывод в консоли**
```
Меня зовут Anna, мне 32 лет
Меня зовут Tom, мне 36 лет
```

**Как это работает**

- `new Student("Anna", 32)` — создаётся объект и сразу вызывается конструктор
- Конструктор принимает параметры `name` и `age`
- `this.name` — обращается к полю объекта, а `name` — к параметру метода
- `introd()` — собирает строку с информацией о студенте

| Что было раньше | Что теперь |
|-----------------|------------|
| Создавали объект через `new Student()` | Создаём через `new Student("Anna", 32)` |
| Поля заполняли вручную: `one.name = "Anna"` | Поля заполняются автоматически через конструктор |
| Кода больше | Кода меньше, всё в одной строке |

---

## Мои заметки
- Конструктор упрощает создание объектов — не нужно отдельно заполнять поля
- `this` — указывает на текущий объект, помогает отличить поле от параметра

Конструктор вызывается автоматически при new
